# WeatherCast

A beautiful real-time weather forecast app built with Next.js, Tailwind CSS, and Open-Meteo API. Shows current weather, 7-day forecast, live updating time and date.

## Features
- 🌍 Automatic geolocation
- ⏰ Live clock and date
- 🌤️ Current conditions with feels-like
- 📅 7-day forecast
- Responsive dark-themed UI
- No API keys required

## Deployed on Vercel

Visit the live app: [https://weathercast-app.vercel.app](https://weathercast-app.vercel.app) (after deployment)

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Open-Meteo Weather API

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
